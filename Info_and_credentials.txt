Repository per il progetto di gestione e visualizzazione dati su incidenti stradali.
Serve per ricreare rapidamente il gestionale, caricare i csv e visualizzare le dashboard.


Questo repository contiene:
- script sql per creare lo schema dati Oracle (DDL)
- script sql per generare l’applicativo Oracle APEX vuoto (pagine, dashboard, lookup)
- script della vista con i dati puliti e rielaborati
- script delle query dei grafici
- Link al dataset Kaggle (csv): https://www.kaggle.com/datasets/anoopjohny/traffic-crashes-crashes/data


Struttura repository:

app_creation_scripts/
|- A01_DDL_Script.sql – DDL di tutte le tabelle (incluse lookup).
|- A01_Traffic_Accidents_App_Script.sql – Export dell’app APEX “vuota” (pagine, dashboard, griglia interattiva).

(Dopo l’esecuzione di questi due script si ottiene l’app funzionante senza dati)

view_and_charts_scripts/
|- traffic_accidents_vw_create.sql – View con dati puliti, formattati e raggruppati.
|- all_charts_queries.sql – Query usate dai grafici APEX.


Note: 
I dati in csv non vengono importati automaticamente nelle tabelle create a priori dalle DDL.
Per popolare il DB: usare APEX Data Workshop per creare nuove tabelle dai csv,
oppure inserire manualmente i dati nel gestionale funzionante appena creato.
